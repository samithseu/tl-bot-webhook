# Webhook Telegram Bot

A Simple Telegram Bot Created with WebHook and integrating with APIs:

| Command  | Description                           |
| -------- | ------------------------------------- |
| /start   | Greetings!                            |
| /me      | Show Telegram account info.           |
| /quote   | Fetch random quote from Internet.     |
| /joke    | Fetch random joke from Internet.      |
| /ask     | Ask AI about something. (briefly)     |
| /profile | Get user profile photo.               |
| /meme    | Fetch programming meme from Internet. |
| /help    | Show all commands.                    |
